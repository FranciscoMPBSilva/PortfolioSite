IPSLink

IPSLink is a mobile application that helps students stay connected with the Polytechnic Institute of Setúbal.

It provides secure access to profiles, direct messaging, notifications, group discussions, and academic services. Students can schedule meetings, submit requests, check the academic calendar, find department contacts, and navigate the campus with an interactive map — all in one place.

IPSLink makes communication and daily tasks simpler and more efficient for the entire academic community.

Technologies used: Flutter, Dart, Firebase.

UnholyHell

The core objective of the game is to eliminate demons and gather resources to build a new city.

The gameplay is divided into two phases: during the night, players venture out to hunt and defeat enemies,when these foes are slain, they drop materials. Then, in the daytime, players use these materials to construct and expand their town.

Technologies used: Unity, C#.

BusNetwork

This Java-based project focuses on creating an interactive system for managing and displaying a bus transit network.

Users can explore detailed information about stops and routes, run various analyses, and adjust live data including route lengths and durations. The application also features ticket issuance and the ability to export modified datasets.

Visualization is powered by JavaFXSmartGraph, and the source code is comprehensively annotated using Javadoc.

Technologies used: IntelliJ, Java, JavaFX
